Tentang Toko Kasir

Toko Kasir dibuat pertama-tama hanya untuk membangun portfolio saya untuk menjadi seorang web developer, toko kasir ini nantinya akan menjadi sebuah website Open Source yang berarti tersedia untuk umum, dan pengguna memiliki kebebasan untuk memodifikasinya sesuai kebutuhan dengan mendownload file yang sudah disediakan dan mengkonfigurasinya di komputer/hosting masing-masing.
